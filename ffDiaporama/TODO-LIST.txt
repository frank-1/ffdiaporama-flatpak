switch version to
    libav 0.8.6 for Ubuntu
    ffmpeg 2.0 for Windows / openSuse / Gentoo / ArchLinux

French version / Correction de bug :
-Corrige la valeur du champ ffDRevision des projets lorsque ceux-ci n'ont pas encore été sauvegardé

English version / Bug fixes :
-Fixes value of the project ffDRevision field when project was not yet saved

French version / Nouveautés :
-Supprime les millisecondes de l'affichage lors de l'utilisation de la variable %PDD%
-Remplace QT Assistant (fenetre du WIKI) par une fenetre Popup intégrée dans l'application, ce qui permet de réduire la taille des paquets d'installation et permet au WIKI de suivre l'interface (quand le wiki est ouvert, il affiche toujours la page correspondant à la fenetre ou boite de dialogue en cour)

English version / New features :
-Remove display of millisecondes when using %PDD% variable
-Replaces QT Assistant (the WIKI window) by a Popup window integrated into the application, which reduces the size of the installation packages and allows to follow the WIKI interface (when the wiki is open, it always displays the page for the current window or dialog box)
