Changelog depuis le dernier SVN

Code cleaning +

French version / Correction de bug :
-Corrige un crash de l'application quand on ajoute une image par Drag & Drop directement dans la zone des blocs de la boite d'édition des slides si la liste des blocs est vide
-Corrige le positionnement de l'icone verrous pour les blocs verrouillés dans la boite slide
-Modifie le menu contextuel des blocs, dans la boite d'edition des slides pour pouvoir verrouiller ou déverrouiller tous les blocs de la sélection
-Corrige le fonctionnement des verrous dans la boite d'edition des slides : maintenant les modifications apportés a une sélection multiple de blocs sont bien répercutés à tous les blocs verrouillés dans les plans suivants (les modifications ne s'appliquaient qu'au 1er bloc de la sélection)

English version / Bug fixes :
-Fixed an application crash when adding an image by drag & drop directly into the blocks table of the edit slides dialog if block list is empty
-Correct the positioning of the lock icone for blocks locked in the slide box
-Modify the contextual menu of blocks, in the edit slides dialog to be able to lock or unlock all the blocks of the selection
-Correct the functioning of locks in slide edit dialog: now the modifications brought to a multiple selection of blocks are echoed to all the locked blocks in the following plans (the modifications applied only to the 1st block of the selection)

French version / Nouveautés :
-Modifie le fonctionnement du bouton "Nouveau projet" : Maintenant il ouvre la boite de dialogue de propriété du projet
-Ajoute la sélection du format d'image (4:3, 16:9 ou cinema) dans la boite de propriété du projet (La sélection n'est possible qu'en création de nouveau projet)
-Eclate la propriété Speed Wave en 4 propriétés pour la musique (forcé à SPEEDWAVE_SINQUARTER), pour les transitions, pour les animations de blocs et pour les animations d'image (cadrage et filtres)
-Modifie la boite d'options pour permettre la définition des valeurs par défaut des propriétés Speed wave pour les transitions, pour les animations de blocs et pour les animations d'image
-Modifie la boite de propriété du projet pour permettre la définition des propriétés Speed wave pour les transitions, pour les animations de blocs et pour les animations d'image
-Modifie les boites transition, édition de slide et édition d'image pour ajouter la sélection de la courbe de vitesse
-Définie et/ou renomme les courbes de vitesse en 5 courbes : Constant speed, Fast then slow, Slow then fast, Faster and faster et Slower and slower

English version / New features :
-Modifies the operation of the "New project" button: Now open the project property dialog
-Adds the selection of format (4:3, 16:9 or cinema) in the project property dialog (selection is only possible when creating new project)
-Split Speed ​​Wave property into 4 properties for music (forced to SPEEDWAVE_SINQUARTER), for transitions, for blocks animations and image animations (frame and filters)
-Changes the options dialog to allow the setting of default Speed ​​wave properties for transitions, block animations and image animations
-Changes the project properties dialog to allow the setting of Speed ​​wave properties for transitions, block animations and image animations
-Modifies the transition, edit slide and edit image dialog to add selection of speed wave
-Defines and/or renamme speed wave to 5 curves: Constant speed, Fast then slow, Slow then fast, Faster and faster et Slower and slower
