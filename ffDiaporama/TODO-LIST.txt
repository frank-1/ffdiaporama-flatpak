French version / Correction de bug :

-Corrige les problèmes Exiv2 sous Windows avec QT5
-Corrige les problèmes de "tab order" dans la boite de dialogue de propriété du projet
-Corrige les problèmes de mémoire sous Windows/64 bits avec QT 5 (Autorise jusqu'à 2Go de mémoire cache au lieu de seulement 512Ko)
-Corrige les problèmes de son lors de la lecture de video avec ffmpeg 1.2 sous Windows avec QT5
-Corrige les problèmes de son lors de la lecture de video avec QT 5 sous Windows dans la boite de dialogue video
-Corrige l'affichage des chemins (path) sous Windows (affichage des logs et dans la boite de vérification de la configuration)
-Corrige un bug dans la boite d'edition des diapositives (Le bouton ajouté fichier restait enfoncé lorsqu'il était clické)
-Désactive les distributions horizontales et verticales quand le nombre d'objects selectionnés est <3
-Corrige le nom de police pour le style de text par défaut afin qu'il s'affiche correctement dans la boite de dialogue d'édition texte sous Windows
-Corrige une erreur dans le format des fichiers .ffd (l'arbre slot était sous l'arbre transition)

-Ajoute une option dans la boite de configuration pour désactiver les tooltips
-Place le bouton Quitter à droite de tous les onglets dans le menu

-Améliore la lecture des vidéos en mode prévisualisation (ignore les frames inutilisées)

-Crée une nouvelle boite de dialogue contenant un composeur d'image pour créer des modèles de vignettes de projet
-Supprime les valeurs qui sont des valeurs par défaut dans les fichiers .ffd (reduction de la taille des fichiers)
-Ajoute un premier jeux de variables <%variable%> utilisable en tant que texte

 Si cette option n'est pas activée, les champs Titre, Album et Auteur des projets ffDiaporama sont limités à 200 caractères au lieu de 30 caractères.
-Remplace la propriété Année par une date longue (limité à 200 caractères) et une date d'événement dans les propriétés du projet ffDiaporama.
-Ajoute un Alias "clipart" dans l'explorateur de fichier multimédia.
 Cet alias apparait comme premier "disque" dans l'arbre des répertoires et pointe vers un nouveau sous répertoire clipart de ffdiaporama et préremplie ce répertoire avec quelques éléments

English version / New features :
-Adds support to QT5 (involves the partial inclusion of qimageblitz library in the source code of ffDiaporama)

-Place the Exit button to the right of all tabs in the menu
-Changes the About box ... to display the information as text and include contributors in the text
-Enhanced video playback in preview mode (ignore unneeded frame)
-Adds an option to select the model of thumbnail in the project properties dialog and add a button to customize the thumbnail
-Creates a new dialog box containing an image composer to create models of thumbnails project
-Removed value that are default value in .ffd file (reduce file size)
-Added a first set of variable like <%variable%> usable with text
-Add an option in Option dialog to enhance id3v2 compatibility. Defaultly, this option is not set.
 If this option is not set, Title, Album and Author field of ffDiaporama projects are limited to 200 characters instead of 30 characters.
-Replace Year property with a long date (limited to 200 characters) and an event date fields to ffDiaporama project properties.
-Adds Alias ​​"clipart" in the media file browser. This alias appears as the first "disk" in the directory tree and points to a new clipart folder under ffdiaporama and syringe this folder with some elements
-Adds the following options in the option dialog
  -Select short dates format
  -Select thumbnail model to apply by default to new project
  -Check or not thumbnail generation when generations of videos

-Adds a button in the text dialog showing the list of available text variables
