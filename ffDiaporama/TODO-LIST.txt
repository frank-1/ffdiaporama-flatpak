French version / Nouvelles fonctionnalités :
-Change le fonctionnement du racourcie SUPPR dans la boite slide :
    -Si le focus est dans la zone interactive ou la table des blocs => supprime le bloc
    -Si le focus est dans la table des plans => supprime le plan
-Scroll la table des blocs lors des drag & drop et que la souris est amenée hors de la table
-Ajoute le Drag & drop depuis un explorateur de fichier vers la table des blocs de la boite slide
-Ajoute une fonctionnalité de verouillage des propriétés des blocs sur le plan précédent
    - Un bloc vérouillé dans un plan ne peut pas être modifié
    - Les modifications sur un blocs sont répercutés dans les plans suivant tant que le verrouillage est actif
    - Les blocs du 1er plan ne peuvent pas être verrouillés

English version / New features :
-Modify functionning of DEL shortcut in slide dialog :
    -If focus is on interactive zone or in blocks table => remove the block
    -If focus is on shots table => remove the shot
-Scroll the blocks table during drag & drop when mouse is brought outside the table
-Add Drag & drop from a file explorer to the blocks table of the slide dialog
-Add a feature to lock blocks properties to their value in previous shot
    - A locked block in a shot can't be modify
    - Modifications to a block are echoed in the following shots as long as the locking is active
    - Blocks of the first shot can't be lock
