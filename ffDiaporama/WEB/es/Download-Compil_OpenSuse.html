Compilar ffDiaporama para openSUSE

Instalación de dependencias

Dependencias para ejecutar ffDiaporama:

Antes de compilar ffDiaporama, asegúrate que tienes todos los paquetes necesarios.

Para ello, abrimos un terminal y escribimos los siguientes comandos:

zypper in libqt4 libQtWebKit4 ffmpeg libSDL-1_2-0 libSDL_mixer-1_2-0 exiv2

Dependencias para compilar ffDiaporama :

Antes de compilar ffDiaporama, asegúrate que tienes todos los paquetes necesarios para compilar.
Para ello, abre un terminal y escribe los siguientes comandos:

zypper in libqt4-devel libQtWebKit-devel libffmpeg-devel libSDL-devel libSDL_mixer-devel wget

Construir ffDiaporama

Una vez instaladas todas las dependencias, abrir un terminal y escribir los siguientes comandos:

cd ~
mkdir ffDiaporama
cd ffDiaporama
wget -O ffDiaporama.tar.gz http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
tar -xzf ffDiaporama.tar.gz
rm ffDiaporama.tar.gz
cd ffdiaporama_trunk/ffDiaporama
qmake
make
make clean

El resultado, una vez compilado ffDiaporama, se encontrará en ~/ffDiaporama/ffdiaporama_trunk/build

Lanzar ffDiaporama

Introducir el comando : ~/ffDiaporama/ffDiaporama_trunk/build/ffDiaporama,
o bien ejecutar ffDiaporama navegando al directorio ~/ffDiaporama/ffDiaporama_trunk/build,
y entonces hacer click en el ejecutable ffDiaporama
