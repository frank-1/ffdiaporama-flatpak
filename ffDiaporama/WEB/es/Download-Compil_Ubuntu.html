Make ffDiaporama for Ubuntu

Traducción en curso

Installation of requirements

Execution requirements:

Before compiling ffDiaporama, be sure that you have all the necessary packages for its execution.
First, insure MEDIBUNTU repository is enabled on your system using this documentation : http://medibuntu.org/repository.php
Second, open a terminal and enter the following commands:

sudo apt-get install libqtcore4 libqtgui4 libqt4-network phonon libqtwebkit4
sudo apt-get install libavcodec-extra-52 libavformat-extra-52 libavdevice-extra-52 libswscale-extra-0 ffmpeg

Compiling requirements:

Before compiling ffDiaporama, be sure that you have all the necessary packages for the compilation. For that open open a terminal and enter the following commands:

sudo apt-get install build-essential libqt4-dev qt4-dev-tools libqtwebkit-dev
sudo apt-get install libavutil-dev libavformat-dev libavcodec-dev libswscale-dev
sudo apt-get install libsdl-image1.2-dev libsdl-mixer1.2-dev

Make ffDiaporama

Once installed requirements, to open a terminal and to enter the following commands:

cd ~
mkdir ffDiaporama
cd ffDiaporama
wget -O ffDiaporama.tar.gz http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
tar -xzf ffDiaporama.tar.gz
rm ffDiaporama.tar.gz
cd ffdiaporama_trunk/ffDiaporama
qmake-qt4
make
make clean

The compilation result is placed in the directory ~/ffDiaporama/ffdiaporama_trunk/build

Start ffDiaporama

Enter the command : ~/ffDiaporama/ffDiaporama_trunk/build/ffDiaporama
or start ffDiaporama with the file explorer since the directory ~/ffDiaporama/ffDiaporama_trunk/build
