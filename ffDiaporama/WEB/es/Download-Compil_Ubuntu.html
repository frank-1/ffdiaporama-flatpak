Compilar ffDiaporama para Ubuntu

Instalación de dependencias

Dependencias para ejecutar ffDiaporama:

Antes de compilar ffDiaporama, asegúrate que tienes todos los paquetes necesarios.
En primer lugar, asegúrate que los repositorios de MEDIBUNTU están disponibles en tu sistema. Consulta esta documentación : http://medibuntu.org/repository.php
En segundo lugar, abre un terminal y escribe los siguientes comandos:

sudo apt-get install libqtcore4 libqtgui4 libqt4-network phonon libqtwebkit4
sudo apt-get install libavcodec-extra-52 libavformat-extra-52 libavdevice-extra-52 libswscale-extra-0 ffmpeg

Dependencias para compilar ffDiaporama:

Antes de compilar ffDiaporama, asegúrate que tienes todos los paquetes necesarios para compilar. Para ello, abre un terminal y escribe los siguientes comandos:

sudo apt-get install build-essential libqt4-dev qt4-dev-tools libqtwebkit-dev
sudo apt-get install libavutil-dev libavformat-dev libavcodec-dev libswscale-dev
sudo apt-get install libsdl-image1.2-dev libsdl-mixer1.2-dev

Construir ffDiaporama

Una vez instaladas todas las dependencias, abre un terminal y escribe los siguientes comandos:

cd ~
mkdir ffDiaporama
cd ffDiaporama
wget -O ffDiaporama.tar.gz http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
tar -xzf ffDiaporama.tar.gz
rm ffDiaporama.tar.gz
cd ffdiaporama_trunk/ffDiaporama
qmake-qt4
make
make clean

El resultado, una vez compilado ffDiaporama, se encontrará en ~/ffDiaporama/ffdiaporama_trunk/build

Lanzar ffDiaporama

Introducir el comando : ~/ffDiaporama/ffDiaporama_trunk/build/ffDiaporama,
o bien ejecutar ffDiaporama navegando al directorio ~/ffDiaporama/ffDiaporama_trunk/build,
y entonces hacer click en el ejecutable ffDiaporama
