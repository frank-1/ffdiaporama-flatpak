Compilar ffDiaporama para Windows

Instalación de dependencias

FfDiaporama está escrito en Qt y utiliza dos bibliotecas externas.
Para compilarlo hay que tener Qt-Creador versión 4.7.3, ffmpeg y SDL.

Qt para Windows:

Sitio de descargas : http://qt.nokia.com/downloads
Selecccionar Qt SDK version 1.1.1 - en concreto http://get.qt.nokia.com/qtsdk/Qt_SDK_Win_offline_v1_1_1_en.exe
Ejecutar el archivo .exe e instalarlo.

librerías ffmpeg para Windows

Sitio de descargas : http://ffmpeg.zeranoe.com/builds/
Obtener la versión más actualizada en las secciones: 32-bit Builds (Static), 32-bit Builds (Shared) y 32-bit Builds
Descomprimir los siguientes 3 archivos .7z :
  • 32-bit Builds (Static) en un subdirectorio ffmpeg bajo winbuild
  • Copiar archivos .dll de 32-bit Builds (Shared) en el directorio winbuild
  • 32-bit Builds (Dev) en un subdirectorio bajo QT (C:\Qt\ffmpeg-win32-dev)

SDL para Windows:

Sitios de descargas : http://www.libsdl.org/download-1.2.php
Obtener la versión más actualizada de Mingw en la sección "Librerías de desarrollo para Windows"- en concreto : http://www.libsdl.org/release/SDL-devel-1.2.14-mingw32.tar.gz
Descomprimir los siguientes archivos .7z en el directorio Qt.

Descargar el código fuente de ffDiaporama:

Descargar el archivo: http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
Descomprimir el archivo .tar.gz en un directorio

Modificaciones en el archivo ffDiaporama.pro:

Hay que modificar el archivo ffDiaporama.pro para indicar en qué directorios están ffmpeg y SDL. En el archivo de ffDiaporama.pro, modificar este bloque :

win32 { 
  DESTDIR     += ../winbuild/ 
  OBJECTS_DIR += ../winbuild/ 
  MOC_DIR     += ../winbuild/ 
  UI_DIR      += ../winbuild/

  INCLUDEPATH += . \
    C:\Qt\ffmpeg-win32-dev\include \                     #------ ffmpeg library path
    C:\Qt\SDL-1.2.14\include                             #------ SDL library path

  LIBS += -L"C:\Qt\ffmpeg-win32-dev\lib" \               #------ ffmpeg library path
          -L"C:\Qt\SDL-1.2.14\lib" \                     #------ SDL library path
          -lavformat -lavcodec -lavutil -lswscale -lSDL 
}

Las líneas escritas en amarillo corresponden a los directorios que se han de modificar.

Construir ffDiaporama

Hay que ejecutar Qt-Creator, abrir el archivo ffDiaporama.pro e iniciar la compilación.

Lanzar ffDiaporama

Doble click en ffDiaporama.exe en el directorio ffDiaporama\winbuild
