ffDiaporama 1.0 Beta 4

Por ahora, no hay una versión estable de ffDiaporama.
Seguimos trabajando, y la 4ª versión está disponible.

 Fecha del paquete: 11/09/2011

Windows

Los paquetes están diseñados para XP-SP3, Vista y Siete.
Sólo están disponibles versiones para 32 bits. (Pero estas versiones también trabajan en sistemas de 64 bits.)

Versión estándar : ffDiaporama.1.0.beta.3_setup.exe.

Si esta versión no trabaja en su ordenador o si planea usar ffDiaporama en una máquina virtual (Xen, Citrix, VMWare, VirtualBox, etc...) usar mejor la versión sin aceleración gráfica : ffDiaporama.1.0.beta.3_setup_NoA.exe.

Finalmente, existe una versión portátil (diseñado por ) disponible en este sitio web: Framakey

 Linux

Estos paquetes están diseñados para Ubuntu y Fedora, pero también está disponible el código fuente para poder instalarlo en cualquier distribución.

Para instalar las dependencias, pueden ser necesarios algunos repositorios, dependiendo de la distribución:
  • Usuarios de Ubuntu deberán activar el repositorio Medibuntu
  • Usuarios de Fedora deberán activar el repositorio RPM Fusion
