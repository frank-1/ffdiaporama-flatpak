Descargas de ffDiaporama

ffDiaporama 1.0 Beta 4 del 11/09/2011

Por ahora, no hay una versión estable de ffDiaporama.
Seguimos trabajando, y la 4ª versión está disponible.

Windows

Los paquetes están diseñados para XP-SP3, Vista y Siete. Sólo están disponibles versiones para 32 bits, pero estas versiones también trabajan en sistemas de 64 bits..

Versión estándar La versión estándar es la versión que hay que privilegiar si se desea instalar ffDiaporama sobre el sistema Windows.
 Estándar
Versión sin aceleración por hardware Si la versión estándar no funciona correctamente en vuestro ordenador (no responde el lector) o queréis utilizar ffDiaporama en una máquina virtual (Xen, Citrix, VMWare, VirtualBox, etc...) entonces usar mejor la versión sin aceleración por hardware
 Sin aceleración
Versión portátil La versión portátil (sin aceleración por hardaware) diseñada para ejecutarse en una memoria USB con el sistema
Framakey

Linux

Para algunas distribuciones hay paquetes ya preparados, pero la aplicación puede también funcionar en todas las distribuciones si compilamos desde el código fuente.


Ubuntu
 Los paquetes para Ubuntu están diseñados para trabajar con Ubuntu 11.04 Natty y Ubuntu 10.10 Maverick.
Es necesario activar el repositorio de Medibuntu para tener cubiertas todas las dependencias.
 Ubuntu-32 bits,  Ubuntu-64 bits

Fedora
 Los paquetes para Fedora están diseñados para funcionar en Fedora 15 y Fedora 14.
Es necesario activar el repositorio de RPM Fusion para tener cubiertas todas las dependencias.		 Fedora 15-32 bits, Fedora 15-64 bits
Fedora 14-32 bits, Fedora 14-64 bits

OpenSuze
 Los paquetes para OpenSuse están diseñados para funcionar en openSUSE 11.4.
Es necesario activar el repositorio de packman-multimedia para tener cubiertas todas las dependencias. 		openSUSE-32 bits, openSUSE-64 bits,

Seguir los desarrollos de ffDiaporama

Los archivos del código fuente de la versión de desarrollo están disponibles en el SVN, en esta dirección : http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk/
Un paquete comprimido tar.gz contiene todos los archivos del código fuente, y está disponible en esta dirección : http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar

Para más información sobre la evolución actual del desarrollo, para proponernos cambios, o para informar de un bugg, os podéis conectar a uno de los siguientes temas del foro de ffDiaporama :
