You have a problem with the software

In case of problem in the use of the software (crash), restart ffDiaporama since a terminal to see if error messages appear.

If the problem persists, visit and post a message at the ffDiaporama Forum (Please, use French or English only) : http://groups.google.com/group/ffdiaporama
  • By describing us the problem in so clear a way and detailed as possible
  • By joining to your e-mail a copy of the information systems shown in the About dialog box (do a simple copy/past in your message)
