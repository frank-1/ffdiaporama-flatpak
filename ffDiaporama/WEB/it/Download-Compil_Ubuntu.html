Make ffDiaporama per Ubuntu

Installazione dei requisiti

Esecuzione requisiti:

Prima di compilare ffDiaporama, bisogna essere sicuri di avere tutti i pacchetti necessari per la sua esecuzione.
Primo, assicurarsi di avere abilitato il repository MEDIBUNTU sul propio sistema usando questa documentazione : http://medibuntu.org/repository.php
Secondo, apri un terminale e digita i seguenti comandi:

sudo apt-get install libqtcore4 libqtgui4 libqt4-network phonon libqtwebkit4
sudo apt-get install libavcodec-extra-52 libavformat-extra-52 libavdevice-extra-52 libswscale-extra-0 ffmpeg

Requisiti di compilazione:

Prima di compilare ffDiaporama, assicurarsi di avere tutti i pacchetti necessari per la sua compilazione. Per questo, aprire un terminale e digitare i seguenti comandi:

sudo apt-get install build-essential libqt4-dev qt4-dev-tools libqtwebkit-dev
sudo apt-get install libavutil-dev libavformat-dev libavcodec-dev libswscale-dev
sudo apt-get install libsdl-image1.2-dev libsdl-mixer1.2-dev

Make ffDiaporama

Requisiti per l'installazione, aprire un terminale e digitare i seguenti comandi:

cd ~
mkdir ffDiaporama
cd ffDiaporama
wget -O ffDiaporama.tar.gz http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
tar -xzf ffDiaporama.tar.gz
rm ffDiaporama.tar.gz
cd ffdiaporama_trunk/ffDiaporama
qmake-qt4
make
make clean

Il risultato della compilazione viene inserito nella directory ~/ffDiaporama/ffdiaporama_trunk/build

Avviare ffDiaporama

Digitate il comando : ~/ffDiaporama/ffDiaporama_trunk/build/ffDiaporama
o avviare ffDiaporama con l'esplorazione dei file nella directory ~/ffDiaporama/ffDiaporama_trunk/build
