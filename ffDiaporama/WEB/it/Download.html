ffDiaporama 1.0 Beta 4

For the moment, there is still no stable version of ffDiaporama.
Works are in progress, and the fourth beta release is now available.

 Package date: 09/11/2011

Windows

Packages are designed for XP-SP3, Vista and Seven.
Only 32 bits versions are available. (But these versions also works on 64 bits systems.)

Standard version : ffDiaporama.1.0.beta.4_setup.exe.

If this standard version does not work on your computer or if you plan to use ffDiaporama in a virtual machine (Xen, Citrix, VMWare, VirtualBox, etc...) use rather the version without material acceleration : ffDiaporama.1.0.beta.4_setup_NoA.exe.

Finally, a portable version (design for ) is available from Framakey website.

 Linux

Packages are designed for Ubuntu and Fedora, but the application can work on any distribution by making it from sources files.
