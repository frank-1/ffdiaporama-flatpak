ffDiaporama 1.0 Beta 4 del 11/09/2011

Per ora, non c'è ancora una versione stabile di ffDiaporama.
I lavori sono in corso, e la quarta versione beta è disponibile.

Windows

I pacchetti sono stati progettati per XP-SP3, Vista e Seven. Sono disponibili le versioni a 32 bit, ma funzionano anche per i 64 bit.

Versione standard La versione standard è ideale se si desidera installare ffDiaporama sul propio sistema Windows. Standard
Versione senza accelerazione hardware Se la versione standard non funziona correttamente (crash) o se si desidera utilizzare ffDiaporama in una macchina virtuale (Xen, Citrix, VMWare, VirtualBox, ecc ...) utilizzare la versione senza accelerazione hardware.
 No Accelerazione
Versione portatile La versione portatile (senza accelerazione hardware) è progettata per funzionare su chiavette USB con il sistema.
Framakey

Linux

Sono stati progettati pacchetti per alcune distribuzioni, ma l'applicazione può essere eseguita su tutte le distribuzioni in grado di compilare i file sorgenti.


Ubuntu
 I pacchetti per Ubuntu sono stati progettati per funzionare con Ubuntu 11.04 Natty e Ubuntu 10.10 Maverick.
L'installazione richiede l'attivazione dei repository Medibuntu per le dipendenze inclusione.
 Ubuntu-32 bits,  Ubuntu-64 bits

Fedora
 I pacchetti per Fedora sono stati progettati per funzionare con Fedora 15 e Fedora 14.
L'installazione richiede l'attivazione dei repository RPM Fusion per l'esame delle dipendenze.		 Fedora 15-32 bits, Fedora 15-64 bits
Fedora 14-32 bits, Fedora 14-64 bits
