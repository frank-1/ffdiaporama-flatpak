Make ffDiaporama per Windows

Installazione dei requisiti

FfDiaporama è scritto in Qt e usa due librerie esterne.
Per compilarlo è necessario disporre di Qt Creator versione 4.7.3, ffmpeg e SDL.

Qt per Windows:

Pagina per scaricare : http://qt.nokia.com/downloads
Clicca su Qt SDK versione 1.1.1 - difatto http://get.qt.nokia.com/qtsdk/Qt_SDK_Win_offline_v1_1_1_en.exe
Avvia il file .exe e installa tutto.

Librerie ffmpeg per Windows

Pagina per scaricare : http://hawkeye.arrozcru.org/
Prendi la versione rilasciata con la data più alta dalle sezioni 32-bit Builds (Static), 32-bit Builds (Shared) e 32-bit Builds (Dev)
Decomprimi i 3 file .7z :
  • 32-bit Builds (Static) in una sotto directory ffmpeg sotto winbuild
  • Copia il file .dll dal 32-bit Builds (Shared) nella cartella winbuild
  • 32-bit Builds (Dev) in una sotto directory di QT (C:\Qt\ffmpeg-win32-dev)

SDL per Windows:

Pagina per scaricare : http://www.libsdl.org/download-1.2.php
Prendi la versione di Mingw rilasciata con la data più alta nella sezione "Development Libraries for Windows" - difatto : http://www.libsdl.org/release/SDL-devel-1.2.14-mingw32.tar.gz
Decomprimi il file .7z preferibilmente nella directory Qt.

Scarica i sorgenti di ffDiaporama:

File da scaricare : http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
Decomprimi il file .tar.gz in una directory

Modifica il file ffDiaporama.pro:

È necessario modificare il file ffDiaporama.pro per indicare in quale directory è stato inserito ffmpeg e SDL.
Nel file ffDiaporama.pro, modifica questo blocco :

win32 { 
  DESTDIR     += ../winbuild/ 
  OBJECTS_DIR += ../winbuild/ 
  MOC_DIR     += ../winbuild/ 
  UI_DIR      += ../winbuild/

  INCLUDEPATH += . \
    C:\Qt\ffmpeg-win32-dev\include \                     #------ Percorso della libreria ffmpeg
    C:\Qt\SDL-1.2.14\include                             #------ Percorso della libreria SDL

  LIBS += -L"C:\Qt\ffmpeg-win32-dev\lib" \               #------ Percorso della libreria 
          -L"C:\Qt\SDL-1.2.14\lib" \                     #------ Percorso della libreria SDL
          -lavformat -lavcodec -lavutil -lswscale -lSDL 
}

Le linee in giallo corrispondono alle directory che bisogna modificare.

Make ffDiaporama

Tu non devi fare altro che avviare Qt-Creator, aprire il file ffDiaporama.pro e avviare la compilazione.

Avviare ffDiaporama

Doppio click su ffDiaporama.exe presente nella directory ffDiaporama\winbuild
