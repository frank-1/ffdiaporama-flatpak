Make ffDiaporama for Windows

Traduzione in corso

Installation of requirements

FfDiaporama is written in Qt and uses two external library.
To compile it you have to have Qt-Creator release 4.7.3, ffmpeg and SDL.

Qt for Windows:

Download page : http://qt.nokia.com/downloads
Click on Qt SDK version 1.1.1 - in fact http://get.qt.nokia.com/qtsdk/Qt_SDK_Win_offline_v1_1_1_en.exe
Start the .exe file and install all.

librairies ffmpeg pour Windows

Download page : http://hawkeye.arrozcru.org/
Get the more up to date release in the 32-bit Builds (Static), 32-bit Builds (Shared) and 32-bit Builds (Dev) sections
Unzip the 3 .7z file :
  • 32-bit Builds (Static) in a sub directory ffmpeg under winbuild
  • Copy .dll file from 32-bit Builds (Shared) in the winbuild folder
  • 32-bit Builds (Dev) in a sub directory under QT (C:\Qt\ffmpeg-win32-dev)

SDL for Windows:

Download page : http://www.libsdl.org/download-1.2.php
Get the more up to date Mingw release in the "Development Libraries for Windows" section - in fact : http://www.libsdl.org/release/SDL-devel-1.2.14-mingw32.tar.gz
Unzip the .7z file rather in your Qt directory.

Download ffDiaporama sources:

File to download : http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
Unzip the .tar.gz file in a directory

Modification of ffDiaporama.pro file:

You have to modify the ffDiaporama.pro file to indicate in which directory you put ffmpeg and SDL.
In the ffDiaporama.pro file, modify this block :

win32 { 
  DESTDIR     += ../winbuild/ 
  OBJECTS_DIR += ../winbuild/ 
  MOC_DIR     += ../winbuild/ 
  UI_DIR      += ../winbuild/

  INCLUDEPATH += . \
    C:\Qt\ffmpeg-win32-dev\include \                     #------ ffmpeg library path
    C:\Qt\SDL-1.2.14\include                             #------ SDL library path

  LIBS += -L"C:\Qt\ffmpeg-win32-dev\lib" \               #------ ffmpeg library path
          -L"C:\Qt\SDL-1.2.14\lib" \                     #------ SDL library path
          -lavformat -lavcodec -lavutil -lswscale -lSDL 
}

The lines in yellow corresponds to directories to be modified.

Make ffDiaporama

You do not have more than to start Qt-Creator, to open the ffDiaporama.pro file and to start the compilation.

Start ffDiaporama

Double click on ffDiaporama.exe since the ffDiaporama\winbuild directory
