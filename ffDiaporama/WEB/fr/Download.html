Windows

Les paquets sont conçus pour  XP-SP3, Vista et Seven. Seuls des versions 32 bits sont disponibles, mais elles fonctionnent également sur des systèmes 64 bits.

Linux

Pour certaines distribution, des paquets ont été conçus, mais l'application peut fonctionner sur toutes les distributions en la compilant depuis les fichiers sources.


Ubuntu
 Les paquets Ubuntu sont conçus pour fonctionner avec Ubuntu 11.10, 11.04 et 10.10

Ubuntu 11.10 - (Oneric)

Avant d'installer le paquet ffDiaporama, installer les librairies ffmpeg en entrant les commandes suivantes dans un terminal.

sudo apt-get install libavcodec-extra-53 libavformat-extra-53 libavdevice-extra-53 libswscale-extra-2

Ubuntu 11.04 (Natty)

Avant d'installer le paquet ffDiaporama, installer les librairies ffmpeg en entrant les commandes suivantes dans un terminal.

sudo apt-get install libavcodec-extra-52 libavformat-extra-52 libavdevice-extra-52 libswscale-extra-0

Ubuntu 10.10 (Maverick)

Avant d'installer le paquet ffDiaporama :
  • Activer le dépôt Medibuntu pour la prise en compte des dépendances.
  • Installer les librairies ffmpeg en entrant les commandes suivantes dans un terminal.
sudo apt-get install libavcodec-extra-52 libavformat-extra-52 libavdevice-extra-52 libswscale-extra-0
