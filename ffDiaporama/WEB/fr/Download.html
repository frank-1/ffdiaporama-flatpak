ffDiaporama 1.0 Beta 4

Pour le moment, il n'existe pas encore de version stable de ffDiaporama.
Les travaux continus, et la quatrième version beta est maintenant disponible.

 Date des paquets : 11/09/2011

Windows

Les paquets sont conçus pour XP-SP3, Vista et Seven.
Seuls des versions 32 bits sont disponibles. (Mais elles fonctionnent également sur des systèmes 64 bits.)

Version standard  : ffDiaporama.1.0.beta.4_setup.exe.

Si cette version standard ne fonctionne pas correctement sur votre ordinateur (crash du lecteur) ou si vous désirez utiliser ffDiaporama dans une machine virtuelle (Xen, Citrix, VMWare, VirtualBox, etc...) utilisez plutôt la version sans accélération matérielle : ffDiaporama.1.0.beta.4_setup_NoA.exe.

Enfin, une version portable (conçue pour ) est disponible depuis le site de Framakey.

 Linux

Les paquets sont conçus pour Ubuntu et Fedora, mais l'application peut fonctionner sur toutes les distributions (en la compilant depuis les fichiers sources).

Pour l'installation des dépendances, l'activation de certains dépôts peut être nécessaire selon la distribution :
  • Les utilisateurs Ubuntu doivent activer le dépôt Medibuntu
  • Les utilisateurs Fedora doivent activer le dépôt RPM Fusion
