Compiler ffDiaporama pour Fedora

Installation des pré-requis

Pré-requis d'exécution :

Avant de compiler ffDiaporama, assurez vous que vous disposez bien de tous les paquets nécessaires à son exécution.

Tout d'abord : activer le dépot RPM Fusion en suivant la documentation http://rpmfusion.org/Configuration
Puis, en entrant les commandes suivantes depuis un terminal :

su
yum install qt4 qt-webkit ffmpeg SDL SDL_mixer exiv2

Pré-requis de compilation :

Avant de compiler ffDiaporama, assurez vous que vous disposez bien de tous les paquets nécessaires en entrant les commandes suivantes depuis un terminal :

su
yum groupinstall "Development Tools"
yum install qt4-devel qt-webkit-devel ffmpeg-devel SDL-devel SDL_mixer-devel wget

Compilation de ffDiaporama

Une fois les pré-requis installés, ouvrir un terminal et entrer les commandes suivantes :

cd ~
mkdir ffDiaporama
cd ffDiaporama
wget -O ffDiaporama.tar.gz http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
tar -xzf ffDiaporama.tar.gz
rm ffDiaporama.tar.gz
cd ffdiaporama_trunk/ffDiaporama
qmake-qt4
make
make clean

Le résultat de la compilation est placé dans le répertoire ~/ffDiaporama/ffdiaporama_trunk/build

Execution de ffDiaporama

Entrer la commande : ~/ffDiaporama/ffDiaporama_trunk/build/ffDiaporama
ou lancer ffDiaporama avec l'explorateur de fichier depuis le répertoire ~/ffDiaporama/ffDiaporama_trunk/build
