Compiler ffDiaporama pour Windows

Installation des pré-requis

ffDiaporama est écrit en Qt et utilise deux librairies externes.
Pour le compiler vous devez disposer de Qt en version 4.7.3, de ffmpeg et de SDL.

Qt pour Windows :

Page de téléchargement : http://qt.nokia.com/downloads
Clicker sur Qt SDK version 1.1.2 - c'est à dire http://get.qt.nokia.com/qtsdk/Qt_SDK_Win_offline_v1_1_2_en.exe
Lancer le fichier .exe et installer tout

Librairies ffmpeg pour Windows :

Page de téléchargement : http://ffmpeg.zeranoe.com/builds/
Prendre les dernières versions, dans les rubriques 32-bit Builds (Static), 32-bit Builds (Shared) et 32-bit Builds (Dev)
Décompresser les 3 fichiers .7z comme suit :
  • 32-bit Builds (Static) doit être décompressé dans un sous répertoire ffmpeg sous le répertoire winbuild
  • Recopier les fichiers .dll contenu dans 32-bit Builds (Shared) dans le répertoire winbuild
  • 32-bit Builds (Dev) doit être décompressé dans un sous répertoire de QT (C:\Qt\ffmpeg-win32-dev)
Note : Les tests ont été réalisés avec la version git-e844abc (2011-06-05)

SDL pour Windows :

Page de téléchargement : http://www.libsdl.org/download-1.2.php
Prendre la dernière version Mingw dans la rubrique "Development Libraries" pour Windows - c'est à dire : http://www.libsdl.org/release/SDL-devel-1.2.14-mingw32.tar.gz
Décompresser le fichier .tar.gz de préférence dans votre répertoire Qt

Téléchargement des fichiers sources de ffDiaporama :

Fichier à télécharger : http://svn.tuxfamily.org/viewvc.cgi/ffdiaporama_trunk.tar.gz?view=tar
Décompresser le fichier .tar.gz dans un répertoire de votre disque dur

Modification du fichier ffDiaporama.pro :

Vous devez modifier le fichier ffDiaporama.pro pour indiquer dans quel répertoire vous avez mis les librairies ffmpeg et SDL
Dans le fichier ffDiaporama.pro, modifier ce bloc :

win32 { 
  DESTDIR     += ../winbuild/ 
  OBJECTS_DIR += ../winbuild/ 
  MOC_DIR     += ../winbuild/ 
  UI_DIR      += ../winbuild/

  INCLUDEPATH += . \
    C:\Qt\ffmpeg-win32-dev\include \                     #------ ffmpeg library path
    C:\Qt\SDL-1.2.14\include                             #------ SDL library path

  LIBS += -L"C:\Qt\ffmpeg-win32-dev\lib" \               #------ ffmpeg library path
          -L"C:\Qt\SDL-1.2.14\lib" \                     #------ SDL library path
          -lavformat -lavcodec -lavutil -lswscale -lSDL 
}

Les parties en jaune correspondent aux répertoires à modifier.

Compilation de ffDiaporama

Il ne vous reste plus qu'à lancer Qt-Creator, à ouvrir le fichier projet ffDiaporama.pro et à lancer la compilation.

Execution de ffDiaporama

Double clicker sur ffDiaporama.exe depuis le répertoire ffDiaporama\winbuild
