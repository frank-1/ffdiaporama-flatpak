I file di configurazione di ffDiaporama

Due file di configurazione

Per poter funzionare, l'applicazione ha bisogno di due file di configurazione.

Il primo file ffDiaporama.xml è memorizzato nella direcoroey ffDiaporama
Per Windows, questa è C:\Program Files\ffDiaporama
Per Linux, è /usr/share/ffDiaporama
Questo file è globale per tutti gli utenti: contiene le impostazioni di default di ffDiaporama.

Un secondo file ffDiaporama.xml è momorizzato nel suo/sua cartella del profilo utente.
Per Windows, questo è C:\Document and settings\[user]
Per Linux, è ~/.ffDiaporama

Quando l'applicazione è avviata per la prima volta, il file di configurazione utente sarà creato, se non esiste.

Nota: Se ffDiaporama è usato da più utenti sullo stesso PC, ci sarà un file di configurazione per ogni utente.
