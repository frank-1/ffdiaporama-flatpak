Defining the properties of a text block

Traduzione in corso

Introduction

The dialog "Edit text" allows you to modify the text within a block:



This dialog is called from two other dialogs when you click the button :

  • From the slide properties dialog
  • From the background properties dialog

Description of the dialog

First zone at the top

Font Allows you to select the font to be used
Color Allows you to select the text color to be used
Size Allows you to select the character size to be used
Shadow effect Allows you to add an outline or shadow to the text:
No effect :
 Text is shown without any shadow or outline
Outline
 Surrounds the characters with a line in the current shadow color
Shadow...
 Add a shadow in the current shadow color in one of the 4 corners.
Shadow color Allows you to select the color of the shadow or the outline.

The second zone

Sets bold characters
Sets characters in italics
Underline the characters
Horizontal alignment: left
Horizontal alignment: centered
Horizontal alignment: justified
Horizontal alignment: right
Vertical alignment: top
Vertical alignment: middle
Vertical alignment: bottom

The third zone

Input field for text
