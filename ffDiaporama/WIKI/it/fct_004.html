Comandi del menu e della barra degli strumenti

La gestione delle diapositive viene effettuata utilizzando i pulsanti sulla barra degli strumenti.
Questi pulsanti sono raggruppati in quattro schede: File / Progetto / Genera video e Aiuto

I comandi di gestione del progetto

La tabella seguente sintetizza i principali comandi:

Pulsanti barra strumenti
 Comandi menu file Scorciatoie da tastiera
 Azione

Nuovo
 CTRL+N Permette di creare un nuovo progetto vuoto utilizzando le impostazioni predefinite come definito nelle Impostazioni applicazione.
(Se il progetto attuale ha delle modifiche che non sono ancora state salvate, ffDiaporama propone di salvarle.)

Apri
 CTRL+O Permette di aprire un progetto esistente. La finestra di selezione file viene visualizzata e permette di selezionare il progetto da aprire.
(Se il progetto attuale ha delle modifiche che non sono ancora state salvate, ffDiaporama propone di salvarle.)
Remark: if the project contains files which were moved or renamed, ffDiaporama will not manage to open them again. In that case, a dialog appears allowing you to select a new file.
Apri recente
Permette di aprire un progetto recentemente utilizzato. ffDiaporama memorizza un elenco degli ultimi dieci progetti.
(Se il progetto attuale ha delle modifiche che non sono ancora state salvate, ffDiaporama propone di salvarle.)

Salva
 CTRL+S Permette di salvare il progetto corrente in un nuovo file.
Se non hai salvato precedentemente il progetto, una finestra di selezione file verrà mostrata e permetterà di selezionare un file destinazione.

Salva come
Permette di slavare il progetto corrente in un nuovo file differente.
Una finestra di selezione file verrà mostrata e permetterà di selezionare un file destinazione.

Impostazioni
Permette di definire le impostazioni dell'applicazione.
Apparirà la finestra delle Impostazioni Applicazione.

Esci
 CTRL+W Permette di uscire da ffDiaporama.
(Se il progetto attuale ha delle modifiche che non sono ancora state salvate, ffDiaporama propone di salvarle.)


I comandi di gestione della diapositiva

La tabella seguente sintetizza i principali comandi:

Pulsanti barra strumenti
 Comandi menu progetto Scorciatoie da  tastiera Azione

Aggiungi titolo
Consente di aggiungere una diapositiva vuota: la diapositiva viene aggiunta al progetto e viene visualizzata nella linea temporale.

Aggiungi file
 INS Consente di aggiungere un foto o un file video: Una finestra di selezione file viene visualizzata e consente di selezionare uno o più file.
Nota: Ogni file selezionato viene aggiunto al progetto in ordine alfabetico per nome del file o in ordine numerico.
La sequenza dei numeri delle telecamere foto/video vengono presi come sono; utilizzando l'ordine numerico si assicura il corretto ordine cronologico delle diapositive.

Aggiungi progetto
Consente di aggiungere alla diapositiva un progetto esistente alla fine del progetto corrente: una finestra di selezione file verrà mostrata e permetterà di selezionare un file di progetto. Diapositive, settaggi di diapositive, musica e sfondo del progetto saranno aggiunte al progetto corrente.

Taglia
 CTRL+X Consente di tagliare la diapositiva seleziionata (la rimuove dalla linea temporale e la mette negli appunti).
Copia
 CTRL+C Consente di copiare una diapositiva selezionata negli appunti.

Incolla
 CTRL+V Consente di incollare nella diapositiva corrente il contenuto degli appunti e lo inserisce nella linea temporale.

Modifica
 Aprirà un sotto menu che consentira di accedere alle funzioni di modifica:
F5
 Propietà sfondo (Equivalente al doppio click sulla traccia di sfondo)
F6
 Propietà diapositiva (Equivalente al doppio click sulla traccia di montaggio)
F7
 Propietà musicale (Equivalente al doppio click sulla traccia musicale)
F8
 Propietà transizione entrante (Equivalente al doppio click sulla transizione)

Rimuovi
 DEL
 Consente di rimuovere la diapositiva corrente.

Nota: le diapositive vengono inserite o al termine del progetto, o dopo la diapositiva corrente, in base all'opzione "aggiungi le nuove diapositive (alla fine del progetto o dopo la diapositiva corrente)" nella finestra di configurazione.
