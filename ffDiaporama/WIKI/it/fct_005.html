Il file progetto

I files progetto di ffDiaporama sono file XML con estenzione .ffd.
  • Questo file XML non contiene copie di elementi multimediali (immagini, video, musica) che sono integrati nel progetto, ma solo i collegamenti a tali file.
  • Questi collegamenti sono relativi a dove sono salvati questi file nel file .ffd.
  • Questo assicura che se la directory contenente il file di progetto (.FFD), contiene anche gli elementi multimediali (direttamente o in una sotto directory), è possibile spostare e/o rinominare la directory.
  • Un altro vantaggio è che un progetto creato con Linux, si può aprire anche con Windows e viceversa.
