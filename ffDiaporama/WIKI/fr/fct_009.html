Générer des films

Introduction

Contrairement à ce qui se passe avec les musicaux qui sont largement portables d'un appareil à un autre, pour les vidéos, il est nécessaire de jongler entre les différents formats possibles pour s'adapter à l'équipement sur lequel elles seront diffusées.

C'est pour cela que nous vous conseillons de conserver autant que possible vos fichiers projets de montage et de générer les films en fonction de l'équipement.

Pour simplifier le paramétrage des génération, ffDiaporama contient une base d'équipement prévue pour les destinations suivantes :
  • Smartphone et équipement portable
  • Système multimédia (de salon)
  • Pour internet
Il est également possible de définir vous même vos paramètres au travers d'une option avancée.

 Nokia - Symbian S60
 En général, les téléphones Nokia fonctionnant avec le système d'exploitation Symbian et étant équipé d'un appareil photo , sont capables de lire des vidéos. Très souvent (mais il faut le vérifier dans les caractéristiques techniques de l'appareil), ceux munis d'un clavier offrent une résolution de 320x240, alors que ceux munis uniquement d'un écran tactile offrent une résolution de 640x360.
Dans les deux cas, le format des vidéos est : Container : MP4 - Codec vidéo : mpeg4 - Codec audio : mp3
Note : Déposer les fichiers vidéos dans le répertoire "My videos" après avoir connecté le téléphone au PC en mode "Périphérique de masse"

Attention : L'ancien format SD est adapté aux anciens téléviseurs utilisant la technologie des tubes cathodiques. Avec ces tubes, les points n'étaient pas rond mais avales : on parle d'anamorphisme.
Lire une vidéo au format SD, sur un téléviseur moderne en LCD, LED ou Plasma donne des résultats franchement calamiteux : l'image étant particulièrement laide !
Donc, si votre téléviseur est récent et si votre équipement de diffusion le permet, préférer toujours un format HD ou Full-HD à un format SD.
