Les paramètres de la ligne de commande

Syntaxe : ffDiaporama [-lang=<local>] [Fichier]

Paramètre
 Explication
Fichier
 Nom d'un fichier projet à charger au démarrage de l'application
-lang=<local> Force la langue à utiliser. <local> peut prendre les valeurs suivantes :
  • en pour utiliser l'anglais
  • es pour utiliser l'espagnol
  • el pour utiliser le grec
  • fr pour utiliser le français
  • it pour utiliser l'italien
  • ru pour utiliser le russe
