Les fichiers de configuration de ffDiaporama

Deux fichiers de configuration

Pour pouvoir fonctionner, l'application utilise deux fichiers de configuration.

Un premier fichier ffDiaporama.xml est stocké dans le répertoire de ffDiaporama
Sous Windows, dans C:\Program Files\ffDiaporama
Sous Linux dans /usr/share/ffDiaporama)
Ce fichier est global pour tous les utilisateurs : Il contient les paramètres par défaut de ffDiaporama.

Un second fichier ffDiaporama.xml dédié à l'utilisateur est stocké dans le profil de l'utilisateur.
Sous Windows, dans C:\Document and settings\[utilisateur]
Sous Linux, dans ~/.ffDiaporama

Au premier lancement de l'application, le fichier utilisateur est créé, s'il n'existe pas déjà.

Note : Si l'application est utilisée par plusieurs utilisateurs sur un même PC, il y aura un fichier de configuration pour chaque utilisateurs.
