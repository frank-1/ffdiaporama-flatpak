Les fichiers projet

Les fichiers projet de ffDiaporama sont des fichiers XML portant l'extension .ffd.
  • Ces fichiers XML n'embarquent aucune copie des éléments multimédias (image, vidéo, musique) qui sont intégrés dans le projet mais uniquement des liens vers ces fichiers.
  • Ces liens sont relatifs à l'endroit où est stocké le fichier .ffd.
  • Ainsi, si le répertoire contenant le fichier du projet (.ffd) contient également les éléments multimédias ou des sous-répertoires contenant ces éléments multimédias, il est possible de déplacer et/ou renommer le répertoire.
  • Autre avantage, un projet créé sous Linux peut être ouvert sous Windows et inversement.
