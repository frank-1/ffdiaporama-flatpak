Éditer le texte

Introduction

La boite "Éditer le texte" permet de modifier le texte d'un bloc :


Cette boite est appelée depuis boite de dialogue de modification des diapositives lorsque vous cliquez sur le bouton ou lorsque vous double-cliquez sur un bloc de type titre.

Description de la boite de dialogue

Style du texte

Ce groupe de réglage fait l'objet d'une gestion de style. Pour plus d'information sur la gestion des styles, consultez : La gestion des styles


 Permet de sélectionner la police de caractère à utiliser

Permet de définir le style des caractères. Les options suivantes sont disponibles :

Met les caractères en gras
Met les caractères en italique
Souligne les caractères
Taille Permet de sélectionner la taille des caractères
Couleur Permet de sélectionner la couleur du texte


Permet de donner un effet de contour ou d'ombre au texte. La première liste permet de définir l'effet :

Aucun effet
 Le texte est affiché sans aucun effet d'ombre ou de contour
Contour Entoure les caractères avec un trait de la couleur de l'ombre
Ombre en ... Ajoute une ombre de la couleur de l'ombre en la positionnant dans l'un des 4 coins.

La seconde liste permet de sélectionner la couleur de l'ombre ou du contour

Permet de définir l'alignement du texte. Les options suivantes sont disponibles :

Alignement horizontal à gauche
Alignement horizontal au centre
Alignement horizontal justifié
Alignement horizontal à droite
Alignement vertical en haut
Alignement vertical au milieu
Alignement vertical en bas

Style du fond de bloc

Note : Cette zone de réglage n’apparaît que pour les blocs de type texte.

Ce groupe de réglage fait l'objet d'une gestion de style. Pour plus d'information sur la gestion des styles, consultez : La gestion des styles


 Permet de définir le type de fond à utiliser. Les types de fond suivant sont disponibles :
Pas de brosse
 La forme n'aura pas de fond et sera donc totalement transparente.
Brosse unie
 Permet de sélectionner une couleur unie.
Motif
 Permet de sélectionner une forme hachurée.
Dégradé à 2 couleurs
 Permet de sélectionner un dégradé de 2 couleurs
Dégradé à 3 couleurs
 Permet de sélectionner un dégradé de 3 couleurs
Image de la librairie
 Permet de sélectionner une image parmi celle présente dans la librairie


Si le type de fond est positionné sur Brosse unie, Motif ou Dégradé, alors cette ligne apparaît et permet de sélectionner la ou les couleurs à utiliser.
(En fonction du type de fond, il y aura une, deux ou trois couleurs sélectionnables.)

Si le type de fond est positionné sur Motif, alors cette ligne apparaît et permet de sélectionner un type de hachure.

Si le type de fond est positionné sur Dégradé à 2 ou 3 couleurs, alors cette ligne apparaît et permet de sélectionner l'orientation du dégradé.
Si le dégradé est a 3 couleurs, des contrôles supplémentaires permettent de définir la position (en pourcentage) du 3eme points de dégradé.

Si le type de fond est positionné sur Image de la librairie, alors cette ligne apparaît et permet de sélectionner une image.
