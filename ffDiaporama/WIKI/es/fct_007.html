Archivos de configuración de ffDiaporama

Dos archivos de configuración

Para poder trabajar, la aplicación necesita dos archivos de configuración.

El archivo .xml principal de ffDiaporama se almacena en el directorio de ffDiaporama
Para Windows, es C:\Program Files\ffDiaporama
Para Linux, es /usr/share/ffDiaporama
Este archivo es global para todos los usuarios: contiene la configuración predeterminada de la aplicación ffDiaporama.

Un archivo .xml secundario de ffDiaporama de un determinado usuario: se sitúa en su directorio donde guarda el perfil.
Para Windows, es C:\Document and settings\[user]
Para Linux, es ~/.ffDiaporama

Cuando se inicia la aplicación por primera vez, se crea el archivo de configuración en el directorio del usuario, si es que aún no existía.

Nota: Si ffDiaporama es utilizado por varios usuarios en el mismo PC, xistirá un archivo de configuración para cada usuario.
