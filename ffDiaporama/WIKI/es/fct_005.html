Archivos de proyecto de ffDiaporama

Archivo de proyecto

Ls archivos de proyecto de ffDiaporama son archivos XML con la extensión .ffd.
  • Estos archivos XML no contienen copias de elementos multimedia (imagen, vídeo, música) que se incluyen en el proyecto, sino vínculos a estos archivos.
  • Estos enlace on relativos a los archivos a la ruta de los archivo .ffd.
  • Esto asegura que, si el directorio que contiene el proyecto del archivo (.ffd) también contiene los elementos multimedia (directamente o en subdirectorios), es posible mover y/o cambiar el nombre del directorio.
  • Otra ventaja es que un proyecto creado bajo Linux se puede abrir en Windows y viceversa.
