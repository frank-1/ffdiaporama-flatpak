Menu y comandos de la barra de herramientas

La gestión de las diapositivas se realiza mediante los botones de la barra de herramientas
Estos botones están agrupados en cuatro pestañas: Archivo / Proyecto / Render y Ayuda

Comandos para gestionar proyecto

La siguiente tabla resume los comandos principales de gestión:

Botón en barra de herramientas
 Comando del menu Atajo de teclado
 Acción

Nuevo proyecto
 CTRL+N Permite la creación de un nuevo proyecto en blanco, con el valor por defecto de la configuración del proyecto, tal como se define en Configuración del programa.
(Si no ha guardado las modificaciones del proyecto actual, ffDiaporama le pedirá que lo guarde.)

Abrir proyecto
 CTRL+O Permite abrir un proyecto existente. Aparece una ventana para poder seleccionar un archivo de proyecto de ffDiaporama.
(Si no ha guardado las modificaciones del proyecto actual, ffDiaporama le pedirá que lo guarde.)
Abrir reciente
Le permite volver a abrir un proyecto guardado recientemente. ffDiaporama almacena una lista de los últimos diez proyectos.
(Si no ha guardado las modificaciones del proyecto actual, ffDiaporama le pedirá que lo guarde.)

Guardar proyecto
 CTRL+S Permite guardar el proyecto actual en un archivo de proyecto.
Si no ha guardado el proyecto, una ventana de diálogo aparece y nos deja seleccionar un destino para el archivo.

Guardar proyecto como
Permite guardar el proyecto actual en un archivo de proyecto diferente.
Una ventana de diálogo aparece y nos deja seleccionar un destino para el archivo.

Configuración
Permite definir la configuración del programa.
La ventana de diálogo Configuración del programa se abrirá

Salir
 CTRL+W Permite salir de ffDiaporama.
(Si no ha guardado las modificaciones del proyecto actual, ffDiaporama le pedirá que lo guarde..)


Comandos de administración de diapositivas

La siguiente tabla resume los principales comandos de gestión:

Botón en barra de herramientas Comando del menu Atajo de teaclado Acción

Añadir
Permite añadir una diapositiva vacía : la diapositiva se añade al proyecto y se muestra en la línea de tiempo.

Añadir archivos
 INS Permite agregar fotos o archivos de vídeo: aparece una ventana de diálogo para poder seleccionar uno o más archivos.
Nota: Cada archivo seleccionado se añade al proyecto por orden alfabético o en el orden numérico de las tomas.
Las cámaras de fotos/vídeos enumeran los archivos a medida que se van tomando. Usar este orden numérico facilita que el proyecto mantenga estructuradas las tomas.

Añadir proyecto
Permite agregar las diapositivas de un proyecto existente al final del actual proyecto: una ventana de diálogo aparece y nos deja seleccionar un archivo de proyecto. Las diapositivas, los ajustes de diapositivas, la música y los fondos del proyecto se añaden al proyecto actual.

Cortar
 CTRL+X Allows you to cut the current slide (remove it from the timeline and put it in the clipboard).
Copiar
 CTRL+C Permite copiar la diapositiva actual al portapapeles.

Pegar
 CTRL+V Permite pegar la diapositiva actual desde el portapapeles en la línea de tiempo.

Editar
 Se abre un sub-menu que da acceso a las funciones de modificación:
F5
 Propiedad del fondo (Equivalente a hacer doble click en la pista del fondo)
F6
 Propiedad de diapositiva (Equivalente a hacer doble click en la pista de montaje)
F7
 Propiedad de la música (Equivalente a hacer doble click en la pista de múica)
F8
 Propiedad de transición (Equivalente a hacer doble click en la transición)

Eliminar
 DEL
 Permite eliminar la diapositiva actual.

Nota: Las diapositivas se insertan al final del proyecto, o después de la diapositiva actual,
dependiendo de la opción "agregar nuevas diapositivas (al final del proyecto o después de la diapositiva actual)" en el diálogo de la configuración del programa.
