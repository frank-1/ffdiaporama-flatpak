Argumentos en línea de comandos para ffDiaporama

Sintaxis : ffDiaporama [-lang=<local>] [Nombre de archivo]

Configuración
 Explicación
Archivo Nombre de archivo del proyecto, archivo a cargar en el inicio de la aplicación
-lang=<local> Idioma que se usará, <local> puede tomar los valores siguientes:
  • en para usar el Inglés
  • fr para usar el Francés
  • es para usar el Español
  • it para usar el Italiano
