Argumentos en línea de comandos para ffDiaporama

Sintaxis : ffDiaporama [-lang=<local>] [Nombre de archivo]

Configuración
 Explicación
Nombre de archivo Nombre del archivo del proyecto al lanzar la aplicación
-lang=<local> Idioma que se va autilizar. <local> puede tomar los siguientes valores:
  • en para usar en Inglés
  • el para usar en Griego
  • es para usar en Español
  • fr para usar en Francés
  • it para usar en Italiano
  • ru para usar en Ruso
