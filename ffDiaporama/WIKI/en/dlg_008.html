Introduction

This dialog box was two different aspects as you clicked a button of device type or the Advanced button.

If you have click on device type button, the dialog box have two selection zone:
  • The first zone allows to select a category of device
  • The second zone allows to select a model of device
If you have click on the Advanced button, the following dialog box will appear:


Description of the dialog box in advanced mode

File format
 Select the type of file to be used for the render movie, among the proposed list.
Video Standard
 Select the video standard to be used for the render movie, PAL or NTSC.
Image format
 Select the image format to be used for the render movie, among the proposed list.
Video codec
 Select the video codec to be used for the render movie, among the proposed list.
Bit rate
 Select the video compression ratio to use among the proposed list
Audio codec
 Select the audio codec to be used for the render movie, among the proposed list.
Bit rate Select the video compression ratio to use among the proposed list

Render movie

Once the device selected or the advanced parameters selected, one click on the OK button starts the process of generating the video.
During this process, one clicks on the "Cancel" button "stop the process.

Remark :

According to the device model or the advanced parameters which you selected, the generation can set a rather long time.

Generally, for a Full-HD 1080p video :
  • On a PC equipped with 4 fast core (Core i7 for example) : The process will set approximately 2 minutes for every minute of the movie to be generated
  • On a PC equipped with 2 core (Core 2 duo for example) : The process will set approximately 5 minutes for every minute of the movie to be generated
That is from 2 to 5 hours for a movie a 1 hour.

This duration will be much less long for a video in resolution VGA. That's why we recommend you to make your tests and focusing in low resolutions and to generate your Full-HD movies only in last step.
