Container and codec

Introduction

A digital video is a computer file called container.
In this file we find:
  • One or several video tracks. Every track is built using a Video Codec.
  • One or several audio tracks. Every track is built using an Audio Codec.
  • Possibly, one or several subtitles. Every subtitles is built in a subtitle format.
  • Possibly, a definition of chapters.
Limitations :
  • ffDiaporama does not manage either subtitles or definitions of chapters.
  • ffDiaporama generates files including a single video track and a single audio track.
