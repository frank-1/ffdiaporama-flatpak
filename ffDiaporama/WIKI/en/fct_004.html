The commands of the menu and the toolbar

The management commands of the projects

The slide management is made:
  • With the buttons of the toolbar
  • With the functions of the File menu
The following table summarizes the main commands of management:

Toolbar button
 File menu command Action

New
 Allows to create a new blank project by using the default project settings as defined in the Application settings dialog box.
(If current project has modifications which were not saved yet, ffDiaporama suggest to save the current project.)

Open
 Allows to open an existing project. The file selection dialog box appears and allows you to select the project to be opened.
(If current project has modifications which were not saved yet, ffDiaporama suggest to save the current project.)

Save
 Allows to save the current project in a file project.
If you did not still save the project, a file selection dialog box appears and allows you to select a file of destination.

Save as
 Allows to save the current project in another file project.
The file selection dialog box appears and allows you to select a file of destination.

Settings
 Allows to define applications settings.
The Application settings dialog box appears.

Quit
 Allow to exit ffDiaporama.
(If current project has modifications which were not saved yet, ffDiaporama suggest to save the current project.)


The commands of management of slides

The slide management is made:
  • With the buttons of the toolbar
  • With the functions of the Project menu
The following table summarizes the main commands of management:

Toolbar button Project menu command Action

Add title
 Allows to add an empty slide : An empty slide is append to the project and is display in the timeline.

Add files
 Allows to add one or several files of type photo or video : A file selection dialog box appears and allows you to select one or several files.
Note: Each file so selected is added to the project in the alphabetical order of the names of file or in the digital order of shots.
Cameras number pictures/movie as they are taken, using the digital order allows to constitute the project in the chronological order of shots.

Add project
 Allows to add the slides of an existing project, at the end the current project : A file selection dialog box appears and allows you to select a project file. Slides as well as slide settings, music and background of the project to be added are added to the current project.

Cut
 Allows to cut the current slide (remove it from the timeline and put it in the clipboard).
Copy
 Allows to copy the current slide to the clipboard.

Paste
 Allows to paste the slide currently in the clipboard to the timeline.

Remove
 Allows to remove the current slide.

Move to left
 Allows to move the current slide by a notch towards the left.

Move to right
 Allows to move the current slide by a notch towards the right.

Render
 Allows to render the movie of the project. The Render movie dialog box appears.

Note: the insertions of slide are either at the end of the project, or after the current slide, according to the option " added the new slides (at the end of the project or after the current slide) " of the configuration dialog box.
