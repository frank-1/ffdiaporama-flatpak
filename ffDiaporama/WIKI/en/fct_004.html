Menu and toolbar commands

The management of slides is carried out using the buttons on the toolbar
These buttons are grouped in four tabs: File / Project / Render and Help

The project management commands

The following table summarizes the main management commands:

Toolbar button
 File menu command Keyboard shortcut
 Action

New
 CTRL+N Allows to create a new blank project using the default project settings as defined in the Application settings.
(If the current project has unsaved modifications, ffDiaporama prompts you to save it.)

Open
 CTRL+O Allows you to open an existing project. The file selection dialog appears and allows you to select the project to be opened.
(If the current project has unsaved modifications, ffDiaporama prompts you to save it.)
Remark: if the project contains files which were moved or renamed, ffDiaporama will not manage to open them again. In that case, a dialog appears allowing you to select a new file.
Open recent
Allows you to reopen a recently opened project. ffDiaporama stores a list of the last ten projects.
(If the current project has unsaved modifications, ffDiaporama prompts you to save it.)

Save
 CTRL+S Allows you to save the current project in a project file.
If you have not previously saved the project, a file selection dialog appears and allows you to select a destination file.

Save as
Allows you to save the current project in a different file.
The file selection dialog appears and allows you to select a destination file.

Settings
Allows you to define applications settings.
The Application settings dialog appears.

Quit
 CTRL+W Allows you to exit ffDiaporama.
(If the current project has unsaved modifications, ffDiaporama prompts you to save it.)

The slide management commands

The following table summarizes the main management commands:

Toolbar button Project menu command Keyboard shortcut Action

Add title
Allows you to add an empty slide : the slide is appended to the project and is displayed in the timeline.

Add files
 INS Allows you to add photo or video files : A file selection dialog appears and allows you to select one or more files.
Note: Each file selected is added to the project alphabetically by filename or in the numerical order of shots.
Cameras number pictures/video sequences as they are taken; using numerical order ensures the correct chronological order of shots.

Add project
Allows you to add the slides of an existing project at the end the current project: a file selection dialog appears and allows you to select a project file. Slides, slide settings, music and background of the project are added to the current project.

Cut
 CTRL+X Allows you to cut the current slide (remove it from the timeline and put it in the clipboard).
Copy
 CTRL+C Allows you to copy the current slide to the clipboard.

Paste
 CTRL+V Allows you to paste the slide currently in the clipboard into the timeline.

Edit
 Opens a sub-menu giving access to the modification functions:
F5
 Background property (Equivalent to a double click on the background track)
F6
 Slide property (Equivalent to a double click on the mounting track)
F7
 Music property (Equivalent to a double click on the music track)
F8
 Entering transition property (Equivalent to a double click on the transition)

Remove
 DEL
 Allows you to remove the current slide.

Note: inserted slides are either at the end of the project, or after the current slide, depending to the option "add the new slides (at the end of the project or after the current slide) " in the configuration dialog.
