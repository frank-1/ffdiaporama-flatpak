Text editor

The dialog box "Edit text" allow to modify the text of a block :


First zone at the top
Font :
Allows to select the font to be used
Color : Allows to select the text color to be used
Size : Allows to select the character size to be used
Shadow effect : Allows to give an effect of outline or shadow to the text:
No effect :
 Text is shown without any effect of shadow or outline
Outline :
 Surround the characters with a line of the color of the shadow
Shadow ... :
 Add a shadow of the color of the shadow by positioning it in one of 4 corners.
Shadow color : Allows to select the color of the shadow or the outline.
The second zone
Puts the bold characters
Puts the characters in italics
Underline the characters
Horizontal alignment to the left
Horizontal alignment to the center
Horizontal alignment tojustify
Horizontal alignment to the right
Vertical alignment at the top
Vertical alignment at the midle
Vertical alignment at the bottom
The third zone
Input field of the text
