The configuration files of ffDiaporama

Two configuration files

To be able to work, the application uses two configuration files.

A first ffDiaporama.xml file is stored in the ffDiaporama directory
For Windows, in C:\Program Files\ffDiaporama
For Linux, in /usr/share/ffDiaporama)
This file is global for all users: it contains the default ffDiaporama application settings.

A second ffDiaporama.xml file dedicated to the user and is stored in his profile folder.
For Windows, in C:\Document and settings\[user]
For Linux, in ~/.ffDiaporama

At the first launch of the application, the user file is created, if it does not already exist.

Note: If ffDiaporama is used by several users on the same PC, there will be a configuration file for every users.
