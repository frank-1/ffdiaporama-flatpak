Define background

Introduction

The "Background properties" dialog box allows to define the settings of the background of the slide:



This dialog box is called in double clicking the background zone of a slide, in the timeline.

Description of the dialog box

Same background as precedent object.
If this option is selected, the slide will have no background definition  but will use the same background as the previous slide:
  • If the previous slide either does not have a background definition, we shall still raise of a notch until a slide have a background definition.
  • If no previous slide have a background definition, then a black frame will be used.
Select a new background
If this option is selected, the slide will have its own background definition.
In that case, the following zones allow to make the settings of the background.

Solid brush Allows to select a plain color.
Gradient 2 colors Allows to select a gradient of 2 colors
Gradient 3 colors Allows to select a gradient of 3 colors
Image from library Allows to select an image among that present in the library
Image disk
 Allows to use an image file.
File is the filename of the actual selected file .
