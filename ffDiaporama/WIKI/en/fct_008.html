ffDiaporama command line arguments

Syntax : ffDiaporama [-lang=<local>] [Filename]

Setting
 Explanation
Filename Filename of the project file to load at application startup
-lang=<local> The language to be used. <local> can take the following values:
  • en to use English
  • fr to use French
  • es to use Spanish
