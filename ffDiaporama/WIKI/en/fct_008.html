The start commands parameters

Syntax : ffDiaporama [-lang=<local>] [Filename]

Setting
 Explanation
Filename Filename of a project file to load at application startup
-lang=<local> Force the language to be used. <local> can take the following values:
  • en to use english
  • fr to use french
