Cutting a video

Introduction

The cut video dialog allows you to define the start and end of the video sequence.



his dialog is called from the Modify slide dialog by clicking on the button  :

Description of the dialog

The progress bar

Displays as follows:
  • The length of the ruler is increased in proportion to the duration of the video.
  • The blue part represents the part of the video which is selected (which will be used in the assembly).
  • The button starts playback. It is then transformed into . Click again on this button to pause.

The working range

  • Positions the cursor at the starting point
  • Uses the current cursor position as starting point (carries out an edit)
  • Positions the cursor at the end point
  • Uses the current cursor position as end point (carries out an edit)
The Volume control allows you to define the " Master Volume " for the file.
