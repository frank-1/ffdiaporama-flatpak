Cut a movie

Introduction

The cut video allows to define the start and end positions to be used for a video.



This dialog box is called since the Modify slide dialog box by clicking on the button  :

Description of the dialogue box

The progress bar

Display respectively:
  • The ruler is increased according to the duration of the video.
  • The blue part represents the part of the video which is selected (which will be used in the assembly).
  • The button allows to start play. It is then transformed there into . Click again on this button to set to pause.

The working range

  • Position the cursor on the start position
  • Use the current cursor position as position of departure (filled the edit zone)
  • Position the cursor on the end position
  • Use the current cursor position as position of stop (filled the edit zone)
Volume allows to define the " Master Volume " for the file.
