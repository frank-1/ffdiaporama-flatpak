Project properties


This dialog allows you to define the information which qualifies the project.
This information will then be registered in the form of TAGs in the rendered video files.

Remarks:

  • The properties of the project are a partial implementation of the MPEG-7 standard on the descriptors of multimedia contents.
  • These properties are written into the video files during the rendering and can be exploited by indexing systems (collection managers) or by jukeboxes.


The Project tab

Settable fields

Fields Remark
Project geometry This field allows you to select the geometry to be used for the project.
Remark: This field is only editable when the dialog box is used for creating a new project. Subsequently, it is not possible to change this value.
Title The title of the project (For more compatibility with the standard id3v2, this fields is limited to 30 characters)
Author The name of the author of the project (For more compatibility with the standard id3v2, this fields is limited to 30 characters)
Remark: This field is often shown by indexing system and jukeboxes under the term Artist
Album The album of the project (For more compatibility with the standard id3v2, this fields is limited to 30 characters)
Year Year (4 numbers)
Remark: This field is different from "timestamp" field
Language The language of the audio track (ISO 639 standard - 3 characters)
Comment A free comment which can spread out on several lines.
Remark: numerous indexing systems or jukeboxes will read only the 1st line.

Remarks:

  • Some fields are automatically filled and do not appear in this dialog:
    Timestamp Date and time of generation of the video.
    Composer Contains the version of ffDiaporama with which the video was produced.
    Encoder Contains the version of libavformat (ffmpeg) with which the video was produced.
  • For more compatibility with indexing systems and jukeboxes, ffDiaporama does not include the whole video TAGs but is closer rather to the standard id3v2 than is the most used by these systems today. Furthermore, the video formats do not support every TAG in the same way.

Chapters

Chapters defined in the project are listed for consultation. (To modify chapters use Chapter button in modify slide dialog.)

The following fields appear:
Fields Remark
# Chapter number
Slide Slide number in wich chapter is defined
Title Chapter title
Start Chapter start position
End Chapter end position
Duration Chapter duration


Remark: Video formats do not support all the chapters in the same way:
Format Remark
3GP/MP4/MKV Full support
AVI/MPG/FLV/OGV/WEBM No support


The option tab

Options for speed waves

These options sets the value of the velocity curves for use whenever the corresponding options are set to "Project Default"
Transition Defined default speed curve for transitions
Block animations Defined default speed curve for block animations
Image animations Defined default speed curve for image animations
