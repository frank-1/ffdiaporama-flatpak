Introduction

The "Slide properties" dialog box allows to define the settings of the slide:



This dialog box is called in double clicking the assembly zone of a slide, in the timeline

Description of the dialogue box

This dialog box includes numerous zones:
  • Zones with left correspond to the settings of the slide and shots
  • Zones with rights correspond to the settings of blocks

To the left, at the top: the shots zone

The shot zone is constituted by a toolbar and a timeline.

The shot toolbar

Add a shot
 Append a shot to end of the timeline
Remove shot
 Remove current selected shot
To left
 Change the order of the shot by moving the current selected shot, of a notch towards the left
To right
 Change the order of the shot by moving the current selected shot, of a notch towards the right
TV Margin
 Enable or disable rulers corresponding to the margins of television sets.

The shot timeline

The shot timeline shows the shots as thumbnail, in the order in which they will be played (of the left towards the right).
For every shot, the thumbnail shows:
  • The shot duration
  • An image representing the shot at the end of its duration

To the left, in the middle: the composition zone

The composition zone allows to move and to re-size interactively blocks, for the current selected shot.
The blocks of the composition are shown with a frame (It allows to differentiate them to graphic elements resulting from the background).
The current selected block is shows with controls handles (small square appearing in corners).
  • Move the controls handles  by means of the mouse to re-size the block
  • Move the block to move it on the slide
Remarks:
If the TV margins are enabled, rulers corresponding to these margins are shown and the mouse will be attracted towards these guides.
This is valid as well for the blocks movements as for the movements of the resizing handles.

In ffDiaporama, the TV margins are of type "Title safe area" and are defined on 90% in width and 90% in height, for all the geometries of project.
If needed, this point can be the object of improvement in the future versions of software!

To the left, below: the shot time setting zone

Shot duration:
 Indicate here the duration for the current selected shot.
For the last shot of the slide, the notion of minimal duration is present.
The following rules apply to calculate the minimal duration of the shot:
  • The total slide duration cannot be lower than the sum of the durations of the entering transition of the slide + the duration of the transition entering of the following slide
  • The total slide duration has to allow to play in its entirety the longest video (The time of the breaks is taken into account)
If you indicate a duration lower than the minimum calculated duration for the shot, it is this minimum duration that will be used on the place of the value which you will have indicate.
Total slide duration:
 The total slide duration is display here.
This duration takes into account possible adjustment of minimal duration of the shot.

To the left, quite below: the slide setting zone

Name of the slide: indicate here a name for the slide. It is this name which will appear on the thumbnail in the main window timeline.

To the right at the top: the block zone

The block zone is established by a toolbar and by a block table.

The toolbar

Add title
 Add a new title block
Add file
 Add a new block of type image or video. A file selection dialogue box appears allowing you to select a file (it is not possible to select several files.)
Remive
 Remove the current block
Copy
 Copy the current block to the ffDiaporama clipboard.
The copy is made with the parameters of block such as they are defined for the current selected shot.
Cut
 Copy the current block to the ffDiaporama clipboard and remove it from the slide
The copy is made with the parameters of block such as they are defined for the current selected shot.
Paste
 Paste block from the ffDiaporama clipboard and insert it.
Up
 Change the order of blocks by moving the active block upward (towards the background)
Down
 Change the order of blocks by moving the active block downward (towards the foreground)

Remarks on blocks:
  • All the blocks, that they are of type title, image or video possess a text.

  • During the insertion of a new block (Add title, Add file or Paste), the block is inserted:
    • In visible mode for the current shot and for all the following shots
    • In hidden mode for the shots located before the current shot
    • Always in last position in the list of blocks (in the foreground)

The block table

The blocks table lists, from top to bottom, the present blocks in the slide.
Blocks are displayed on 3 columns:
  • The first column indicates the type of block (=Title, =Image, =Video)
  • The second column indicates the block which plays the sound during the shot : a icon appears for the block which has the sound
  • The third column indicates:
    • The text of the block, if it is about a block of type titles
    • The name of the file, if it is about a block of type image or video
The active block is the current selected block in the block table. To change active block, simply select a block in the list or in the composition zone.

Remarks on blocks:
  • Order of blocks can change a shot in the other one
  • Hidden blocks are shown in grey tints and italics

A droite en bas : L'onglet Bloc

Cet onglet permet d'effectuer des réglages sur la taille et la position du bloc actif et de son contenu.


 Cliquer sur ce bouton pour rendre visible ou invisible le bloc actif.
Note : Les blocs vidéos invisibles sont automatiquement mis en pause
Ce bouton n'est actif que pour des vidéos.
Cliquer sur ce bouton pour donner le son, pendant ce plan, au bloc vidéo actuellement actif
Cliquer sur le bouton Recadrage et correction pour appeler la boite de dialogue Framing and correction of image or movie dialog box

Géométrie : Permet de sélectionner la contrainte géométrique à appliquer au bloc.
Trois type de contrainte sont disponibles :
  • Géométrie du projet : Le bloc sera contraint à la géométrie du projet (4/3, 16/9 ou 40/17)
  • Géométrie de l'image : Le bloc sera contraint à la géométrie d'origine de l'image ou de la vidéo
  • Géométrie libre : Le bloc ne sera pas contraint
Représente la position horizontal du bloc de texte à partir de la gauche.
Cette position est exprimée en pourcentage (%) de la largeur total de l'image.
Représente la position vertical du bloc de texte à partir du haut de l'image.
Cette position est exprimée en pourcentage (%) de la hauteur total de l'image.
Représente la largeur du bloc de texte.
Cette position est exprimée en pourcentage (%) de la largeur total de l'image.
Représente la hauteur du bloc de texte.
Cette position est exprimée en pourcentage (%) de la hauteur total de l'image.
Rotation selon l'axe des X exprimée en degré.
Rotation selon l'axe des Y exprimée en degré.
Rotation selon l'axe des Z exprimée en degré.

A droite en bas : L'onglet Forme

Cet onglet permet d'effectuer des réglages sur la forme du bloc actif.

Forme
 Permet de sélectionner une forme pour le bloc. La forme sélectionnée servira :
  • de masque de fusion pour les images et les vidéos.
  • de zone de coupure pour les textes
Bordure
 Épaisseur de la bordure :

Définie l'épaisseur de la bordure (Cette valeur est exprimée en 1/1080eme de la hauteur de l'écran, c'est à dire qu'une valeur de 10 correspond à 10 points sur un écran FullHD 1920x1080, mais seulement à 6 points sur un écran 1280x720).

Couleur de bordure :
Définie la couleur du trait à utiliser pour dessiner la bordure

Type de trait de bordure:
 Définie le type de trait (plein, pointillé, etc...)
Ombre
 Position de l'ombre :
 Définie dans quel coin l'ombre sera positionnée (Note : La position de l'ombre est indépendante des rotations du bloc : Le coin inférieur droit est toujours le coin inférieur droit quelque soit les rotations appliquées au bloc)

Distance de l'ombre Définie la distance entre le bloc et son ombre
Opacité
 Opacité
 Définie l'opacité du bloc. Cette valeur s'applique également au texte du bloc.

A droite en bas : L'onglet objet


Cet onglet permet d'effectuer des réglages sur le contenu du bloc.

Cliquer sur ce bouton pour appeler la boite Define the properties of a text block dialog box.
Cliquer sur ce bouton pour appeler la boite Apply filters to an image or a video dialog box
Ce bouton n'est actif que pour les objets contenant des images ou des vidéos.
Cliquer sur ce bouton pour appeler la boite Cut a video dialog box
Ce bouton n'est actif que pour les objets contenant des vidéos.

En fonction du type d'objet, différents contenus sont possibles :
  • Pour une image ou une vidéo : Le nom du fichier avec un bouton de sélection de fichier permettant de changer de fichier
    • Attention, si vous remplacer le fichier (image ou vidéo) par un autre dont la géométrie d'image n'est pas la même, vous devrez revoir le cadrage de l'image dans tous les plans.
  • Pour un titre : Définition d'une brosse pour la forme :

    Pas de brosse :
    		 La forme n'aura pas de fond et sera donc totalement transparente.
    Brosse unie :
    		 Permet de sélectionner une couleur unie.
    Motif : Permet de sélectionner :
    • Une forme de hachure parmi l'une des douze disponibles
    • Une couleur pour les lignes de hachure
    Dégradé à 2 couleurs : Permet de sélectionner un dégradé de 2 couleurs
    Dégradé à 3 couleurs : Permet de sélectionner un dégradé de 3 couleurs
    Image de la librairie : Permet de sélectionner une image parmi celle présente dans la librairie

A droit en bas : L'onglet information

Affiche des informations sur le bloc :
  • Si le bloc est un titre : affiche que c'est un titre
  • Si le bloc est une image : affiche les informations EXIF contenues dans l'image
  • Si le bloc est une vidéo : affiche les informations de format de la vidéo
