The project file of ffDiaporama

The project file

Files project of ffDiaporama are XML files carrying the .ffd extension.
  • This file XML contains no copy of the multimedia elements (image, video, music) which are joined into the project but only the links towards these files.
  • These links are relative to the place where is stored the .ffd file.
  • So, if the directory containing the project file (.ffd ) also contains the multimedia elements or sub-directories containing these multimedia elements, it is possible to move and/or to rename the directory.
  • Other advantage, a project created under Linux can be opened under Windows and conversely.
